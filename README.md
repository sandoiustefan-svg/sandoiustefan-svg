## 🚀 Selected Public Projects

### 🌬️ Wind Power Forecasting
**End-to-end machine learning pipeline** for **61-hour ahead regional wind power forecasting in Norway**.

- Integration of **MET weather forecast and nowcast datasets**
- Feature engineering with temporal and regional alignment pipelines
- Deep learning forecasting models (GRU / LSTM / Transformer architectures)
- Modular ML pipeline with reproducible experiments
- **FastAPI service** for model inference and prediction serving
- Automated workflows and experiment setup

📂 Repository: `ForecastingWindPower`

---

### 🔊 Vehicle Sound Classification
Audio-based **vehicle type classification system** (8 classes) using deep learning.

- Production-grade audio preprocessing pipeline (librosa-based)
- Spectrograms, MFCCs, chroma features
- CNN, RNN, and hybrid model architectures
- FastAPI inference API

📂 Repository: `vehicle-sound-classification`

---

### 📈 Stock Price Forecasting
Hybrid **ARIMA–LSTM time-series forecasting system** inspired by the M3 forecasting
competition.

- Per-series decomposition into trend, seasonality, and residuals
- Statistical and deep learning model comparison
- Modular pipeline design and evaluation
- Accompanied by a **research-style paper**

📂 Repository: `stock-price-forecasting`
